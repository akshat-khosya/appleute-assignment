# Getting Started

Follows these steps to start the code
## Client

For the client: 
1.  ```cd client```
2. ```npm i ```
3. ```npm start```
## Server
For the Server 
1.  ```cd client```
2. Check the .env.example file to setup env variables
3. ```npm i ```
4. ```npm start```
